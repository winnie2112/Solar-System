### Documentation
- [Docs](https://www.electronjs.org/)
- [Install Node](https://nodejs.org/en/download/)

### Run the app after installation
Check if Node.js was installed correctly:
- node -v
- npm -v
Run the app in terminal
- cd <Path_to_Solar_System app>
- npm start
